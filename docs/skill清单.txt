
## Skills 目录使用场景和描述

### 1. **algorithmic-art** (算法艺术)
**使用场景**: 当你想创建生成式艺术、流场动画、粒子系统或其他基于代码的艺术作品时使用
**描述**: 使用 p5.js 创建算法艺术,支持种子随机性和交互式参数探索。先创建算法哲学,然后通过代码表达为可交互的 HTML 艺术品。适合创建独特的视觉艺术、动态图案和生成式设计。

### 2. **brand-guidelines** (品牌指南)
**使用场景**: 需要为任何类型的作品应用 Anthropic 官方品牌颜色和排版时使用
**描述**: 提供 Anthropic 的官方品牌标识和样式资源,包括颜色方案(深色、浅色、强调色)和字体(Poppins 用于标题,Lora 用于正文)。可以将品牌风格应用到演示文稿、文档或任何需要统一视觉识别的作品中。

### 3. **canvas-design** (画布设计)
**使用场景**: 需要创建海报、艺术作品、设计图或其他静态视觉作品时使用
**描述**: 创建博物馆或杂志级别的视觉艺术,输出为 PNG 或 PDF 文件。先创建设计哲学,然后在画布上表达为精美的视觉作品。强调极简文字、空间表达和专家级工艺,适合创建高质量的平面设计作品。

### 4. **doc-coauthoring** (文档协作)
**使用场景**: 需要编写文档、提案、技术规范、决策文档或类似结构化内容时使用
**描述**: 提供结构化的文档协作工作流程,包括三个阶段:上下文收集、细化与结构化、读者测试。帮助用户高效地传递上下文、通过迭代完善内容,并验证文档对读者的有效性。

### 5. **docx** (Word 文档处理)
**使用场景**: 需要创建、编辑或分析 .docx 文件,包括跟踪更改、添加评论、格式化等操作时使用
**描述**: 全面的 Word 文档处理工具包。支持创建新文档(使用 docx-js)、编辑现有文档(使用 OOXML)、文本提取、跟踪更改(红线工作流)和评论管理。适合处理专业文档、法律文件和需要版本控制的文档。

### 6. **frontend-design** (前端设计)
**使用场景**: 需要构建网页组件、页面、应用程序或任何 Web UI 时使用
**描述**: 创建独特的、生产级别的前端界面,具有高设计质量。避免通用的 AI 美学,强调大胆的美学方向、独特的字体选择、创意的颜色主题和意想不到的布局。支持 HTML/CSS/JS、React、Vue 等技术栈。

### 7. **internal-comms** (内部沟通)
**使用场景**: 需要编写各种内部沟通文档时使用,如状态报告、领导层更新、3P 更新、公司通讯、FAQ 等
**描述**: 提供一套资源来帮助编写各种内部沟通文档。包含多种格式的指南和示例,如 3P 更新(进展、计划、问题)、公司通讯、FAQ 回答等,确保沟通符合公司的格式和风格要求。

### 8. **mcp-builder** (MCP 构建器)
**使用场景**: 需要创建或管理 Model Context Protocol (MCP) 服务器时使用
**描述**: 用于构建和管理 MCP 服务器的工具包。(注意:此 skill 的详细内容未在提供的文件中展开)

### 9. **pdf** (PDF 处理)
**使用场景**: 需要提取文本和表格、创建新 PDF、合并/拆分文档或填写 PDF 表单时使用
**描述**: 全面的 PDF 处理工具包。支持文本和表格提取、PDF 创建、合并/拆分、旋转页面、添加水印、密码保护和表单填写。使用 pypdf、pdfplumber、reportlab 等 Python 库,以及 qpdf、pdftotext 等命令行工具。

### 10. **pptx** (PowerPoint 处理)
**使用场景**: 需要创建、编辑或分析 .pptx 演示文稿时使用
**描述**: 全面的演示文稿处理工具包。支持从 HTML 创建新演示文稿(使用 html2pptx)、使用模板创建演示文稿、编辑现有幻灯片(使用 OOXML)、文本提取和幻灯片转换为图像。包含缩略图生成、幻灯片重排和文本替换等实用工具。

### 11. **skill-creator** (技能创建器)
**使用场景**: 需要创建新的 skill 或更新现有 skill 以扩展 Claude 的能力时使用
**描述**: 提供创建有效 skill 的指南。包括 skill 的核心原则(简洁、适当的自由度)、skill 的结构(SKILL.md、脚本、参考资料、资源)、创建流程(理解、规划、初始化、编辑、打包、迭代)和最佳实践。

### 12. **slack-gif-creator** (Slack GIF 创建器)
**使用场景**: 需要创建针对 Slack 优化的动画 GIF 时使用
**描述**: 提供创建 Slack 优化动画 GIF 的工具和知识。包括 Slack 要求(尺寸、FPS、颜色、时长)、核心工作流(GIFBuilder)、绘图图形(使用 PIL)、动画概念(抖动、脉冲、弹跳、旋转、淡入淡出等)和优化策略。

### 13. **theme-factory** (主题工厂)
**使用场景**: 需要为幻灯片、文档、报告、HTML 页面等应用主题样式时使用
**描述**: 提供 10 个预设主题的工具包,每个主题都有精心选择的颜色调色板和字体配对。可以将主题应用到任何作品,或根据需要生成新的自定义主题。主题包括海洋深处、日落大道、森林树冠、现代极简等。

### 14. **web-artifacts-builder** (Web 工件构建器)
**使用场景**: 需要构建 Web 工件或组件时使用
**描述**: 用于构建 Web 工件的工具包。(注意:此 skill 的详细内容未在提供的文件中展开)

### 15. **webapp-testing** (Web 应用测试)
**使用场景**: 需要与本地 Web 应用程序交互和测试时使用,包括验证前端功能、调试 UI 行为、捕获浏览器截图和查看浏览器日志
**描述**: 使用 Playwright 进行 Web 应用程序测试的工具包。提供服务器生命周期管理(支持多个服务器)、静态 HTML 和动态 Web 应用的测试方法、侦察-然后-行动模式,以及最佳实践和示例。

### 16. **xlsx** (Excel 处理)
**使用场景**: 需要创建、编辑或分析 Excel 电子表格(.xlsx、.xlsm、.csv、.tsv 等)时使用
**描述**: 全面的电子表格处理工具包。支持创建带公式和格式的新电子表格、读取或分析数据、修改现有电子表格同时保留公式、数据分析和可视化,以及公式重新计算。特别强调财务模型的颜色编码标准和公式构建规则。
